# Grid란?

Grid 레이아웃은 두 방향 레이아웃 시스템이다.

![0](/images/post/css/grid-layout/0.png)

![1](/images/post/css/grid-layout/1.png)

- 그리드 컨테이너(Grid Container): display: grid를 적용하는 Grid의 전체 영역이다.
    
    Grid 컨테이너 안의 요소들이 Grid 규칙의 영향을 받아 정렬된다.
    
- 그리드 아이템(Grid Item): Grid 컨테이너의 자식 요소들이다.
    
    이 아이템들이 Grid 규칙에 의해 배치된다.
    
- 그리드 트랙(Grid Track): Grid의 행(Row) 또는 열(Column)
- 그리드 셀(Grid Cell): Grid의 한칸을 가리키는 말이다. div태그같은 실제 html 요소는 그리드 아이템이고, 이 그리드 아이템 하나가 들어가는 “가상의 칸”이 그리드 셀이다.
    
- 그리드 라인(Grid Line): Grid 셀을 구분하는 선이다.
- 그리드 번호(Grid Number): Grid 라인의 각 번호다.
- 그리드 갭(Grid Gap): Grid 셀 사이의 간격이다.
- 그리드 영역(Grid Area): Grid 라인으로 둘러싸인 사각형 영역으로 그리드 셀의 집합이다.

Grid의 속성은 두가지로 나뉜다.

- 컨테이너에 적용하는 속성
- 아이템에 적용하는 속성
    
    ![2](/images/post/css/grid-layout/2.png)
    

### display: grid

Grid 컨테이너에 적용하는 속성

```css
.grid-container {
	display: grid;
}
```

### grid-template-columns

컨테이너에 적용하는 속성으로 Grid 트랙의 열 너비와 갯수를 지정하는 속성

```css
.grid-container {
	grid-template-columns: 200px 200px 500px;
	grid-template-columns: 1fr 1fr 1fr;
	grid-template-columns: repeat(3, 1fr);
	grid-template-columns: 200px 1fr;
	grid-template-columns: 100px 200px auto;
	grid-template-columns: minmax(100px, auto);
	grid-template-columns: repeat(auto-fill, minmax(20%, auto));
	grid-template-columns: repeat(auto-fit, minmax(20%, auto));
}
```

- 여러가지 단위를 사용할 수 있고 섞어서 쓸 수 있다.
- 200px 200px 500px은 column을 200px, 200px, 500px로 만든다는 의미
- fr은 fraction(분수, 파편)이라는 의미로 숫자 비율대로 트랙의 크기를 나눈다. 1fr 1fr 1fr은 1:1:1 비율인 3개의 column을 만든다는 의미
- repeat함수는 repeat(반복횟수, 반복값) 이다.
- 200px 1fr은 왼쪽의 첫번째 column은 200px로 고정되고 나머지 두번째 column은 1의 비율로 유연하게 움직인다.
- minmax함수는 minmax(최소값, 최대값) 이며 최소값 만큼의 크기는 확보하고 내용이 많아 최소값을 넘기면 최대값까지만 늘어나도록 하는 설정이다. auto로 하면 알아서 늘어난다.
- auto-fill은 column의 개수를 미리 정하지 않고 설정된 너비가 허용하는 한 최대한 셀을 채운다.
    
    ![3](/images/post/css/grid-layout/3.png)
    
- auto-fit은 auto-fill과 유사하지만 남는공간을 채워준다는 차이점이 있다.
    
    ![4](/images/post/css/grid-layout/4.png)
    

### grid-template-rows

Grid 트랙의 행 높이와 갯수를 지정하는 속성

### gap

그리드 셀 사이의 간격을 설정한다.

```css
.container {
	row-gap: 10px;
	/* row의 간격을 10px로 */
	column-gap: 20px;
	/* column의 간격을 20px로 */
	gap: 10px 20px;
	/* row-gap: 10px; column-gap: 20px; */
	gap: 20px;
	/* row-gap: 20px; column-gap: 20px; */
}
```

- row-gap: row의 간격 조정
- column-gap: column의 간격 조정
- gap: 순서대로 두개 쓰면 row, column의 간격 조정
- gap: 하나만 쓰면 row와 column의 간격조정

### grid-row-end

아이템에 적용하는 속성으로 각 셀의 영역을 지정한다.

![5](/images/post/css/grid-layout/5.png)

```css
(빨간색 영역)
.item:nth-child(1) {
	grid-column-start: 1;
	grid-column-end: 3;
	grid-row-start: 1;
	grid-row-end: 2;
}
```

- grid-column-start: 아이템의 column 시작번호를 지정하는 속성
- grid-column-end: 아이템의 column 끝번호를 지정하는 속성
- grid-row-start: 아이템의 row 시작번호를 지정하는 속성
- grid-row-end: 아이템의 row 끝번호를 지정하는 속성

단축해서 쓰면

```css
(빨간색 영역 단축버전)
.item:nth-child(1) {
	grid-column: 1 / 3;
	grid-row: 1 / 2;
}
```
